# Основы Express.js
Дополнение к видео курсу
Каждый урок находится в своей ветке
Переключение веток происходит по приципу:
```sh
$ git checkout step-N
```
где N - номер урока.
То есть урок №1:
```sh
$ git checkout step-1
```
Скачать зависимости 
```sh
$ npm i
```
